# EmployeeManager

**EmployeeManager** — это приложение на Java для управления информацией о сотрудниках.
---
## Функциональность

    Добавление, редактирование и удаление записей о сотрудниках
    Просмотр списка сотрудников
    Поиск сотрудников по различным критериям

## Установка

    Клонируйте репозиторий:

git clone https://github.com/ваш_пользователь/EmployeeManager.git

Перейдите в директорию проекта:

cd EmployeeManager

Соберите проект с помощью Gradle:

    ./gradlew build

## Использование

После сборки проекта вы можете запустить приложение командой:

java -jar build/libs/EmployeeManager.jar

## Tребования

    Java 11 или новее
    Gradle 6.0 или новее
